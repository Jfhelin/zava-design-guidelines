# Zava Page Structure

Use this file when recreating page layout patterns that should immediately read as Zava-inspired.

## Overall Composition

- page background should be a light gray-blue neutral
- top of page should use a dark teal header region
- white content cards should sit on the light page background
- footer should return to the dark teal brand surface

## Header Pattern

- dark teal banner as the outer header shell
- inner navigation area can use a light surface with rounded corners
- logo sits left
- one simple auth or account action sits right
- hero copy sits below navigation inside the dark header area

## Hero Pattern

- concise heading
- short supporting paragraph
- supporting links may be inline and light colored
- avoid giant illustrations or oversized CTA clusters

## Body Pattern

- section heading in dark text
- filter and search controls grouped near the heading
- product or content cards arranged in a clean responsive grid
- cards should be white with subtle borders and modest corner radius

## Card Pattern

- white background
- subtle border
- modest radius, approximately 5px to 10px on the current site
- no dramatic shadow required
- title, muted description, prominent price or value row, then action

## Footer Pattern

- dark teal footer background
- brand description in first column
- multiple compact link columns
- footer headings use light blue accents
- footer links use muted warm-gray text
- final copyright row separated visually below
